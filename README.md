# Covid-19-Dashboard-Analysis
An interactive COVID-19 dashboard that visualizes global pandemic trends, vaccination progress, and country-level comparisons through dynamic and time-series visualizations using R Shiny.

## Objectives

* Analyze global COVID-19 trends over time
* Compare cases, deaths, and vaccination rates across countries
* Evaluate the impact of vaccination on pandemic indicators
* Explore correlations between health and socioeconomic factors
* Provide focused insights on Jordan and regional patterns

## Libraries Used

R Shiny dashboard built using key libraries including:
**shiny, dplyr, ggplot2, plotly, reshape2, lubridate, RColorBrewer, DT, tidyr, scales, ggrepel, corrplot.**

## Dataset
The dashboard is built using publicly available COVID-19 datasets containing global time-series records of confirmed cases, deaths, vaccinations, testing rates, and policy indicators.
The dataset used in this project is publicly available on Kaggle:
https://www.kaggle.com/datasets/georgesaavedra/covid19-dataset/data
