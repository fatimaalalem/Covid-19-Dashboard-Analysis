# Covid-19-Dashboard-Analysis
An interactive COVID-19 dashboard that visualizes global pandemic trends, vaccination progress, and country-level comparisons through dynamic and time-series visualizations using R Shiny.

## Objectives

* Analyze global COVID-19 trends over time
* Compare cases, deaths, and vaccination rates across countries
* Evaluate the impact of vaccination on pandemic indicators
* Explore correlations between health and socioeconomic factors
* Provide focused insights on Jordan and regional patterns


## Dashboard Features / Sections

### Global Overview

* Vaccination vs Deaths Analysis
* Global Correlation Heatmap
* Interactive Choropleth Map
* Climate & Seasonality Trends

---

### Continental Analysis

* Weekly Cases by Continent
* Total Cases Comparison
* Vaccination Distribution (Box Plots)
* Best vs Worst Performing Countries

---

### Socioeconomic Factors

* GDP vs Testing Rates
* Population Density vs Cases/Deaths
* Expected vs Observed Deaths
* Income Tier Comparisons
* Socioeconomic Correlation Analysis

---

### Policy & Epidemiological Dynamics

* Stringency Index vs Case Rates
* Early vs Late Response Impact
* Positivity Rate Analysis
* Vaccination Effect on Mortality



### Jordan Focused Analysis

* Cases vs Death Waves
* Jordan vs Neighboring Countries
* Policy & Vaccination Impact
* National Correlation Heatmap



## Libraries Used

R Shiny dashboard built using key libraries including:
**shiny, dplyr, ggplot2, plotly, reshape2, lubridate, RColorBrewer, DT, tidyr, scales, ggrepel, corrplot.**
